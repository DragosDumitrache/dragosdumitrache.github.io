# dragos.dumitrache.dev

Personal site and blog. Staff Engineer & Engineering Manager writing about backend systems, infrastructure, developer experience, and whatever else is on my mind.

Built with [Astro](https://astro.build) using the [Dotfiles](https://github.com/nabsiddiqui/dotfiles-astro-theme) theme.

## Running locally

```bash
npm install
npm run dev
```

## Deployment

Deployed to GitHub Pages via GitHub Actions on push to `main`.
